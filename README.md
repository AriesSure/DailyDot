# DailyDot - Habit Tracking & Achievement Card Application

## Overview

DailyDot is a modern Flask-based habit tracking and personal achievement application.
It helps users build positive habits, track daily progress, and create beautiful achievement cards to celebrate their journey.
Featuring a clean, responsive design and intuitive interface, DailyDot combines productivity with motivation.

## Key Features

### Habit & Task Management

- **Habit Tracking** – Create, edit, and monitor habits with customizable icons and time periods
- **Daily Check-ins** – Mark habits as complete with notes and timestamps
- **Progress Insights** – View completion rates and streaks
- **Todo Management** – Create and manage daily tasks with due dates and reminders

### Achievement Card System

- **Daily Cards** – Unique cards for each day with custom images and quotes
- **Smart Image Integration** – Auto-fetch images via Pexels API with fallback to Lorem Picsum
- **Quote Categories** – Motivational, success, persistence, growth, and daily inspiration
- **Personalization** – Add notes and habit completion counts to your cards

### Analytics & Statistics

- **Habit Statistics** – Track completion rates and check-in counts
- **Weekly/Monthly Views** – Calendar-style progress tracking
- **Annual Overview** – Year-long habit visualization
- **Todo Productivity** – Monitor task completion trends

### User Management

- **Secure Authentication** – User accounts with password policies
- **Profile Controls** – Change passwords and manage settings
- **Data Privacy** – User-specific data isolation and security

## Tech Stack

- **Backend**: Flask 2.3.3 (Python 3.8+)
- **Database**: SQLite + SQLAlchemy ORM
- **Auth**: Flask-Login
- **Forms**: Flask-WTF (CSRF protection)
- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Icons**: Font Awesome 6.5.0
- **Migrations**: Flask-Migrate + Alembic
